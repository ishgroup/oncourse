To install on an OS X machine do this:

1) Append 	127.0.0.1 scc.local.oncourse.net.au	
	$ sudo vi /etc/hosts

2) Point DocumentRoot and Directory sections of ish-local.conf to resource project.

Example: 
		DocumentRoot /Users/anton/willow/code/resources/resources
		<Directory "/Users/anton/willow/code/resources/resources">
		    Options Indexes FollowSymLinks
		    AllowOverride All
		    Order allow,deny
		    Allow from 127.0.0.1
		</Directory>
	
3)
	$ sudo cp ish-local.conf /etc/apache2/other/ish-local.conf
	$ sudo /usr/sbin/apachectl restart
